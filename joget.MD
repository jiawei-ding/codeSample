### joget调研

- 简介

  Joget是一个开源的低代码平台，旨在帮助开发人员和企业快速构建和部署Web应用程序和工作流程。以下是一些关于Joget的主要特点和功能：

  1. **低代码开发：** Joget提供了一个直观的可视化界面，使开发人员能够使用拖放方式轻松创建应用程序和工作流程，而无需编写大量的代码。
  2. **工作流程管理：** Joget支持创建和管理复杂的工作流程，用于自动化业务流程。它具有强大的工作流设计器，允许定义任务、审批、通知等工作流元素。
  3. **表单构建器：** 开发人员可以使用Joget的表单设计器创建自定义表单，这些表单可以与应用程序和工作流程集成。表单可以包括各种字段类型，如文本、日期、附件等。
  4. **数据集成：** Joget允许与各种数据源集成，包括数据库、Web服务、REST API等。这使得应用程序可以访问和操作不同来源的数据。
  5. **权限和安全性：** Joget提供了丰富的权限和安全性控制，允许管理员定义谁可以访问应用程序和工作流程，以及谁可以执行特定操作。
  6. **多语言支持：** Joget支持多语言界面，使应用程序可以本地化以满足不同地区和语言的需求。
  7. **插件和扩展性：** Joget具有丰富的插件架构，允许开发人员编写自定义插件来扩展平台的功能。
  8. **部署灵活性：** Joget支持云部署和本地部署，可以在多种环境中使用，包括云、本地服务器和容器。
  9. **社区支持：** Joget拥有一个活跃的开发和用户社区，提供文档、教程和支持，有助于开发人员和组织更好地使用平台。

  Joget的低代码特性使得开发人员可以更快速地构建企业级应用程序，而无需深入的编码知识。它在各种行业和应用中都有广泛的应用，包括工作流程自动化、客户关系管理（CRM）、项目管理、人力资源管理（HRM）等。 Joget可用于加速应用程序开发并提高业务流程的效率。



- joget的历史，发展历程

  oget是一个开源低代码平台，它的历史和发展经过了多个阶段，下面是Joget的主要历史和发展过程：

  1. **创立初期（2009年）：** Joget最初于2009年由Raveesh Dewan在新加坡创立。它是一个开源的工作流和表单构建工具，旨在帮助企业快速开发和部署自定义工作流程和应用程序。
  2. **开源发布（2010年）：** 在2010年，Joget宣布开源，并在Google Code上发布源代码。这使得开发人员和组织能够自由使用、修改和扩展平台。
  3. **Joget Workflow的发布（2011年）：** 2011年，Joget Workflow 3.0版本发布，引入了强大的工作流引擎和表单构建器。这个版本开始吸引更多的开发人员和企业用户。
  4. **持续改进和增强（2012年 - 现在）：** 自2012年以来，Joget持续进行改进和增强。平台不断更新，引入了更多功能，如移动应用支持、多语言支持、插件扩展性、集成能力等。这些改进使Joget逐渐发展成为一个全面的低代码开发平台。
  5. **Joget DX的发布（2019年）：** 2019年，Joget发布了Joget DX，这是一个全新的版本，带来了一系列改进和创新，包括全新的用户界面、增强的低代码开发工具、企业级应用程序支持等。Joget DX进一步提高了平台的易用性和功能性。
  6. **全球用户和社区（2020年 - 现在）：** 随着时间的推移，Joget在全球范围内积累了大量的用户和开发者社区。它在多个行业和领域中被广泛使用，包括政府、医疗保健、金融、制造业、教育等。
  7. **合作伙伴和生态系统（2020年 - 现在）：** Joget建立了众多合作伙伴关系，包括系统集成商、独立软件供应商（ISV）和解决方案提供商，以扩展其生态系统并提供更多解决方案。

  Joget的发展历程显示出其持续关注用户需求和市场趋势，不断改进和增强平台，以满足不同行业和组织的需求。作为一个开源低代码平台，Joget继续在全球范围内拥有强大的社区和用户基础，为各种企业应用程序的快速开发提供了有力支持。



- joget有哪些产品

  Joget 提供了一系列的产品和解决方案，以满足不同用户和组织的需求。以下是 Joget 的主要产品和解决方案，并对每个产品进行详细介绍：

1. **Joget DX（Digital Transformation平台）：**
   - **简介：** Joget DX 是 Joget 的核心产品，是一个低代码数字转型平台，旨在帮助组织实现数字化转型、构建应用程序和自动化工作流程。它结合了低代码开发的便捷性和工作流程管理的强大功能。
   - **主要特点：**
     - 可视化低代码开发：提供可视化的界面，允许用户快速创建自定义应用程序和工作流程。
     - 强大的工作流程引擎：支持复杂的工作流程设计和管理，包括任务、审批、通知等。
     - 自定义表单构建：具备表单设计器，允许用户创建各种表单，包括文本、日期、附件等。
     - 数据集成：支持与不同数据源的集成，包括数据库、Web服务、REST API等。
     - 权限和安全性：提供丰富的权限和安全性控制，确保数据和应用程序的安全。
     - 扩展性和插件支持：具有插件架构，允许开发人员扩展平台的功能。
2. **Joget Workflow（工作流程管理）：**
   - **简介：** Joget Workflow 是早期的 Joget 产品，它专注于工作流程管理和自动化。虽然在 Joget DX 中包括了工作流程管理功能，但 Joget Workflow 仍然可用并受支持。
   - **主要特点：**
     - 强大的工作流程引擎：支持多种工作流程设计和自定义任务配置。
     - 表单构建：允许创建自定义表单，用于工作流程的输入和输出。
     - 集成：支持与其他系统和数据源的集成，以实现自动化流程。
3. **Joget Cloud（云服务）：**
   - **简介：** Joget Cloud 是一项托管服务，为用户提供了在云中部署和管理 Joget 应用程序的解决方案。它减少了用户的基础设施管理工作，使他们可以专注于应用程序的开发和维护。
   - **主要特点：**
     - 云托管：将 Joget 应用程序托管在云中，无需用户自行配置和管理服务器。
     - 自动伸缩：根据负载需求自动伸缩，以确保高可用性和性能。
     - 安全性：提供数据安全性和隐私保护的措施。
4. **Joget Marketplace（市场）：**
   - **简介：** Joget Marketplace 是一个在线市场，提供了各种用于扩展 Joget 平台功能的插件、应用程序模板和解决方案。
   - **主要特点：**
     - 插件和模板：提供各种插件和模板，可以用于快速扩展和定制 Joget 应用程序。
     - 应用程序示例：提供了各种行业和应用领域的示例应用程序，用于加速应用程序开发。
     - 社区贡献：允许 Joget 社区成员分享和贡献他们的插件和模板。

总之，Joget 提供了一系列产品和解决方案，旨在帮助组织加速应用程序开发、数字化转型和工作流程自动化。用户可以根据其需求选择合适的产品和解决方案，并根据需要扩展和定制平台。 Joget DX 是 Joget 的核心产品，提供了低代码数字转型的全面解决方案。



- 介绍joget DX的技术架构

  Joget DX 的技术架构是为构建灵活、可扩展且易于维护的低代码平台而设计的。以下是 Joget DX 的主要技术架构要点：

  1. **前端技术：**
     - **HTML5/CSS3：** Joget DX 的用户界面采用现代的 HTML5 和 CSS3 技术，以确保跨平台和响应式设计。
     - **JavaScript：** 前端交互和用户体验通过 JavaScript 处理，包括表单验证、动态界面更新等。
     - **AngularJS：** Joget DX 使用 AngularJS 框架来构建富客户端应用程序，提供了强大的前端功能和可扩展性。
  2. **后端技术：**
     - **Java：** Joget DX 的后端核心是基于 Java 的，这使得平台具有跨平台性和强大的性能。
     - **Spring Boot：** Joget DX 使用 Spring Boot 框架来构建快速、可靠的应用程序后端。Spring Boot 提供了自动配置和快速开发的能力。
     - **Hibernate：** Joget DX 使用 Hibernate 作为数据访问框架，用于管理和操作数据存储。
     - **RESTful API：** 所有的应用程序和工作流程都可以通过 RESTful API 进行访问和集成，这提供了开放的接口和互操作性。
     - **插件架构：** Joget DX 具有可扩展的插件架构，允许开发人员编写自定义插件以扩展平台的功能。这些插件可以包括自定义表单元素、工作流程引擎扩展、集成器等。
  3. **数据库：**
     - Joget DX 支持多种数据库，包括 MySQL、PostgreSQL、Microsoft SQL Server 等。用户可以根据需求选择适合他们组织的数据库。
  4. **安全性：**
     - Joget DX 重视安全性，提供了用户身份验证和授权的机制，以及数据安全性的控制。
     - 用户身份验证可以通过内置的用户管理和集成LDAP、Active Directory等方式实现。
     - 数据安全性得到保护，确保只有授权的用户可以访问和操作敏感数据。
  5. **工作流程引擎：**
     - Joget DX 集成了一个强大的工作流程引擎，用于创建、管理和执行工作流程。它支持任务分配、审批、通知、状态管理等工作流程相关功能。
     - 工作流程引擎的核心是基于 BPMN 2.0（业务流程建模与标记语言）标准的，这意味着可以创建符合行业标准的工作流程。
  6. **多租户支持：**
     - Joget DX 具有多租户支持，允许在单个实例中托管多个独立的租户。每个租户都可以有自己的应用程序和工作流程，但在相同的 Joget DX 实例中运行。
  7. **云和容器支持：**
     - Joget DX 可以轻松部署在云平台上，如AWS、Azure、Google Cloud等，以及容器化平台上，如Docker和Kubernetes。
  8. **插件和模块化架构：**
     - Joget DX 的插件和模块化架构使得平台可以轻松扩展和定制，开发人员可以根据需要编写自定义插件、主题和模块。

  Joget DX 的技术架构被设计为具有高度的可扩展性、可维护性和灵活性，以满足不同组织的需求。这个架构使得 Joget DX 成为一个强大的低代码数字转型平台，适用于各种应用程序开发和工作流程自动化场景。

- 主流的低代码开发平台有哪些

  低代码开发平台是一种帮助开发人员和企业快速构建应用程序的工具，通常通过可视化界面和少量的手动编码来实现。以下是一些主流的低代码开发平台：

  1. **OutSystems：** OutSystems 是一家领先的低代码平台提供商，提供可视化开发工具和一套强大的集成功能，帮助用户快速构建企业级应用程序。
  2. **Microsoft Power Apps：** Microsoft Power Apps 是微软的低代码平台，与Azure和Office 365集成，使用户可以轻松创建自定义应用程序、自动化流程和报告。
  3. **Appian：** Appian 是一家低代码平台提供商，专注于数字流程自动化和业务流程管理。它提供了可视化流程设计工具和快速应用程序开发功能。
  4. **Mendix：** Mendix 是一家低代码平台公司，提供用于构建自定义应用程序的工具和一流的应用程序生命周期管理功能。
  5. **Salesforce Lightning Platform：** Salesforce Lightning Platform（之前称为Force.com）是Salesforce的低代码平台，用于创建和扩展Salesforce应用程序。
  6. **Joget DX：** Joget DX 是一个开源的低代码数字转型平台，提供可视化开发、工作流程管理和应用程序构建功能。
  7. **Zoho Creator：** Zoho Creator 是Zoho的低代码开发工具，用于构建自定义应用程序，适用于小型企业和中小型企业。
  8. **Quick Base：** Quick Base 是一个低代码平台，用于构建自定义应用程序、自动化流程和报表。
  9. **AppSheet：** AppSheet 是Google Cloud的低代码开发平台，用于构建自定义移动应用程序和自动化工作流程。
  10. **K2：** K2是一个数字流程自动化平台，提供可视化流程设计和低代码应用程序开发。

  这些是一些主要的低代码开发平台，每个平台都有其独特的特点和优势，适用于不同的应用场景和组织需求。选择最适合你的低代码平台取决于你的项目需求、技术栈和预算。
